# Products API

## Description

Products API is a simple RESTful API built with Express.js that allows users to manage a collection of products. It provides functionality to retrieve all products, create new products, and retrieve individual products by their ID.

## Getting Started

To get started with the Products API, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/ZainabSameer/Testing-Express-APIs-with-Jest-Project.git
   cd Testing-Express-APIs-with-Jest-Project

2. **Install the dependencies**:
   ```bash
   npm install

3. **Run the application**:
   ```bash
   npm run dev
   
##### The server will run on http://localhost:3001.

## Interact with the API:

* Get all products: GET /products
* Create a new product: POST /products with a JSON body containing productName, price, description, and category.
* Get a product by ID: GET /products/:id
  
## Technologies Used

* Node.js
